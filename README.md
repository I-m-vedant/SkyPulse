# ⚡ SkyPulse — Weather Dashboard

A multi-feature weather web app built with pure HTML, CSS, and Vanilla JavaScript.

## ✨ Features
- 🔍 Search weather by city name
- 📍 Auto-detect location using Geolocation API
- 🌡️ Current weather details (temp, humidity, pressure, feels like)
- 📅 5-day forecast with day names and icons
- 🎨 Dynamic backgrounds based on weather condition (Clear, Rain, Clouds, Snow, Thunderstorm)
- 🕐 Recent searches saved with localStorage (clickable chips)
- ⚠️ Error handling for invalid city names

## 🛠️ Tech Used
- Vanilla JavaScript (no frameworks)
- HTML5 & CSS3
- OpenWeatherMap API
- Geolocation API
- localStorage

## 📚 Key JS Concepts Used
- fetch() + Promise chains (.then)
- Destructuring
- Array methods (filter, map, forEach, slice, unshift)
- DOM manipulation
- Event delegation
- localStorage for persistent state
- Reusable functions + template literals

## 🚀 How to Run Locally
1. Clone this repo or download the files
2. Get a free API key from [openweathermap.org](https://openweathermap.org/api)
3. Replace `apiKey` in `script.js` with your own key
4. Open `index.html` in your browser
